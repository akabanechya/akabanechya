HTML 5 语法和基础标签

一，HTML5 基本骨架

向浏览器声明该文档的类型是 HTML 5

HTML 5 文档的头部区域开始标记，lang=”zh-CN” 为声明当前页面使用的语言受众，通常针对搜索引擎。

meta 为元数据，即描述数据的数据，charset=”UTF-8” 为声明当前网页使用的字符集为 UTF-8。

name=”Keywords” 为元数据名称，content=”” 为元数据内容。<meta name=”Keywords” content=””> 为设置当前网页的关键词信息。

<meta name=”Description” content=””> 为设置当前网页的描述信息。

<title>Name</title> 为设置当前网页标题为 Name。

</head> HTML 5 文档的头部区域结束标记。

<body> HTML 5 文档的主体区域开始标记。

</body> HTML 5 文档主体区域的结束标记。

二，列表标签

一，无序列表 （unordered list）

  • unordered list （无序列表：一种列表格式，其中项目没有特定的顺序或优先级。）
  1. ul 具备一个 type 属性，有三个属性值，默认 disc（实心圆），circle（空心圆），square（实心方块），该属性在 html 5 中已被废弃。

二，无序列表 （ordered list）

  • ordered list 有序列表：按照一定顺序排列的一组项目或元素。
  1. ul 具备一个 type 属性，有五个属性值，默认 1（表示数字编号），a（表示小写英文字母），A（表示大写英文字母），i（表示小写罗马数字），I（表示大写罗马数字）
  2. ul 具备一个 star 属性，用来规定起始值。
  3. ul 具备一个 reversed 属性，用来规定是否倒序。
